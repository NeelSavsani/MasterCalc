<h1 >🧮 MasterCalc Pro</h1>

<p >
  A modern and responsive <b>All-in-One Calculator Website</b> built using HTML, CSS, and JavaScript.
</p>

<p >
    🌐 Live Demo
  <a href="https://mastercalcpro.vercel.app/" target="_blank">

<h2>✨ Features</h2>

<h3>🔢 Basic Calculator</h3>
<ul>
  <li>Addition</li>
  <li>Subtraction</li>
  <li>Multiplication</li>
  <li>Division</li>
  <li>Real-time calculation display</li>
  <li>Keyboard support</li>
</ul>

<h3>📐 Trigonometry Calculator</h3>
<ul>
  <li>Sine (sin)</li>
  <li>Cosine (cos)</li>
  <li>Tangent (tan)</li>
  <li>Angle-based calculations</li>
</ul>

<h3>💰 EMI Calculator</h3>
<ul>
  <li>Loan amount calculation</li>
  <li>Interest rate support</li>
  <li>Monthly EMI estimation</li>
  <li>Total payable amount</li>
</ul>

<h3>🧾 GST Calculator</h3>
<ul>
  <li>GST addition</li>
  <li>GST removal</li>
  <li>Percentage-based tax calculations</li>
</ul>

<h3>💾 Data Size Converter</h3>
<ul>
  <li>Bytes</li>
  <li>KB</li>
  <li>MB</li>
  <li>GB</li>
  <li>TB</li>
</ul>

<h3>📏 Unit Converter</h3>
<ul>
  <li>Length</li>
  <li>Weight</li>
  <li>Distance</li>
  <li>Speed</li>
  <li>Time</li>
  <li>Volume</li>
</ul>

<h3>🌡️ Temperature Converter</h3>
<ul>
  <li>Celsius</li>
  <li>Fahrenheit</li>
  <li>Kelvin</li>
</ul>

<h3>📐 Geometry Calculator</h3>
<ul>
  <li>Area</li>
  <li>Perimeter</li>
  <li>Volume</li>
  <li>Circle calculations</li>
  <li>Triangle calculations</li>
  <li>Rectangle calculations</li>
</ul>

<h3>🎂 Age Calculator</h3>
<ul>
  <li>Exact age calculation</li>
  <li>Years / Months / Days output</li>
  <li>Date difference calculation</li>
</ul>
